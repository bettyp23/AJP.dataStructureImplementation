## Overview

This project implements four fundamental data structures in Java:
- **MyLinkedList**: A singly linked list with add, remove, and display operations
- **MyStack**: A stack implementation with push, pop, peek, and isEmpty operations
- **MyQueue**: A queue implementation with enqueue, dequeue, peek, and isEmpty operations
- **BinarySearchTree**: A binary search tree with insert, contains, and in-order traversal operations

## Compile & Run

To compile all Java files:
```bash
javac *.java
```

To run the demonstration:
```bash
java DataStructureDemo
```
